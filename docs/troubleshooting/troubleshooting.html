This section provides in-depth troubleshooting information about common problems of SIL Kit simulations. Most issues can be identified based on log messages by the application.

Note

To troubleshoot any problem involving the SIL Kit, activate the built-in logging on all involved applications. It is recommended to write the participant log to a file with the most detailed log level Trace:

Logging:
  Sinks:
  - Type: File
    Level: Trace
    LogName: ParticipantLog
