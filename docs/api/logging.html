Logging Service API

SIL Kit provides a Logging service that can be used for writing log messages of specified log levels to files, stdout or distribute them with remote logging to other simulation participants. The logger can be obtained by calling GetLogger(). Refer to Logging Configuration for how to configure the logging service.
